# 为什么教材生涩难懂?
- 我无意间刷到了关于"知识的诅咒"视频, 大概指的是**认知偏差**,当人有了某个知识时,会下意识的认为其他人也有相同的知识.
- 我个人认为并不是教材有**防自学**, 而是写教材的人中了**知识的诅咒**, 完全默认为: 我们已经明白的这个知识, 完全忘记了当初时如何从**0到1**的过程, 这也是为什么老师经常说:"我都已经讲了八百遍了!!!" 家长教不好孩子的原因了, 很熟悉吧, 就和知识的诅咒有关.
